Will I Reply?
=============

Project for Stanford's CS 221. Machine learning algorithm that predicts whether you'll respond to a given email.

This repository considers training data both from the Enron dataset, and your own personal Gmail.

The repository structure modeled after this `template <https://drivendata.github.io/cookiecutter-data-science/>`_ from Driven Data.
